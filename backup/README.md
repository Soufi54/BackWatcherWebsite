# Backup — main website page

This folder contains a copy of the main landing page (`index.html`) as of the backup date.

- **index.html** — snapshot of the homepage before modifications.
- To restore this variant as the live page, copy `backup/index.html` over the root `index.html`.

Asset paths in the backup (CSS, JS, images) point to the project root; the live site continues to use the root `index.html`.
